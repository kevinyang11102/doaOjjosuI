# 健身房管理系统 java73

## 项目概述

本项目是基于Java与SpringBoot框架，结合Vue进行前端开发实现的健身房管理系统。系统采用前后端分离的架构模式，提供了一套完整且含有详细注释的源码，以及相应的技术栈说明文档。

## 技术栈

- **架构模式：** B/S架构，MVC设计模式
- **开发环境：**
  - IDE: IntelliJ IDEA
  - JDK版本: 1.8
  - 构建工具: Maven
  - 数据库: MySQL 8.0/5.7
- **系统环境：** 支持 Windows / Mac 操作系统

## 功能模块

### 系统角色

- **管理员：**
  - 会员管理
  - 健身房管理（含课程管理、器材管理）
  - 员工管理
  - 报名表管理
- **会员：**
  - 用户登录（系统访问入口）
  - 可视化图表查看数据
  - 课程查询
  - 在线充值
  - 充值记录查询
  - Vip管理
  - 购买记录查询
  - 我的资料查看与维护（修改头像、修改密码）

## 运行环境

- **操作系统：** Windows / Mac
- **数据库：** MySQL 8.0/5.7

## 部署步骤

1. 确保运行环境满足要求（操作系统、数据库等）
2. 使用IntelliJ IDEA导入项目源码
3. 配置数据库连接
4. 使用Maven构建项目
5. 启动后端服务
6. 前端项目使用Vue CLI启动
